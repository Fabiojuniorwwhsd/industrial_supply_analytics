# Dashboard

Use `data/processed/base_analitica_supply.csv` como base para Power BI ou Looker Studio.

## Sugestão de páginas

### Visão Geral
- Total de pedidos
- OTIF %
- Pedidos em atraso %
- Materiais em estoque crítico %
- Registros de avaria
- Evolução mensal de pedidos

### Fornecedores
- Ranking de fornecedores por atraso
- Ranking de fornecedores por avarias
- OTIF por fornecedor
- Tempo médio de entrega por fornecedor

### Estoque e Materiais
- Materiais em estoque crítico
- Estoque atual vs estoque mínimo
- Categorias com mais divergências
- Categorias com mais avarias

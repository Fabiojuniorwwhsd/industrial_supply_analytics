# Descrição para Currículo

**Industrial Supply Analytics — KPIs de Suprimentos, Estoque e Entregas**  
**Projeto de BI e Análise de Dados | mai/2026**

• Projeto de **Business Intelligence (BI)** e **Análise de Dados / Data Analytics** inspirado em cenário industrial fictício, utilizando dados sintéticos de pedidos, fornecedores, entregas, estoque e avarias para acompanhamento de KPIs operacionais.

• Realizei tratamento, padronização e consolidação dos dados com **Python, SQL, Excel/Sheets e Power Query**, aplicando conceitos de **ETL/ELT, Data Cleaning / Data Quality** e modelagem de dados.

• Desenvolvi indicadores como **OTIF, pedidos em atraso, divergência de quantidade, estoque crítico, avarias por fornecedor e tempo médio de entrega**, apoiando análises de performance e tomada de decisão orientada por dados.

• Estruturei documentação no **GitHub**, incluindo bases fictícias, dicionário de dados, etapas do projeto e evidências visuais do dashboard, preservando confidencialidade e boas práticas relacionadas à LGPD.
